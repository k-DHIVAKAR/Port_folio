HTML

  • Elements: Fundamental building blocks of HTML, enclosed in tags.
  • Tags: Markers defining the beginning and end of elements.
  • Attributes: Provide additional information within tags.
  • Document Structure: Composed of key elements like DOCTYPE,
    html, head, and body.




CSS

  • Selectors: CSS selectors target HTML elements for styling. They can be based on element type, class, ID, or other attributes.
  • Properties: CSS properties define how selected elements should appear. They include attributes like color, font-size, margin, padding, etc.
  • Cascade: CSS follows a cascading order, where multiple styles can apply to an element. The order of precedence is determined by specificity, importance, and source order.
  • Box Model: Every HTML element is represented as a rectangular box in CSS. This model consists of the content area, padding, border, and margin, defining the space an element occupies.
